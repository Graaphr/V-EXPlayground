### How to use

1. clone this repo
```bash
git clone https://github.com/Graaphr/V-EX.git pbl-vex
```
2. get in to main folder
```bash
cd pbl-vex
```
3. get in to folder front-end
```bash
cd front-vex 
```
4. install package
```bash
npm install 
```
5. run react
```bash
npm run dev 
```
6. back 
```bash
cd .. 
```
7. move in backend 
```bash
cd back-vex
```
8. install laravel
```bash
composer install
```
9. install db
```bash
php artisan migrate
```
10. run laravel
```bash
php artisan serve
```

if you happen an error, contact us
